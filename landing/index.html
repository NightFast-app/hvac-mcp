What it does

Refrigerant PT saturation

R-410A, R-32, R-454B, R-22, R-134a. Bubble + dew for zeotropic blends.

Charge diagnosis

Superheat & subcool with TXV / piston diagnosis and restriction detection.

Symptom tree

Ranked probable causes with test procedure + typical fix. Seeded by a 608 Universal tech.

Fault code decoder

Carrier, Trane, Goodman, Lennox, Rheem, York, Mitsubishi, Daikin. Alias-aware (Bryant → Carrier).

Code lookup

IRC / IMC / IPC citations with Florida amendments (HVHZ, condensate, wind anchor).

Pipe + duct sizing

DWV via IPC Table 710.1, supply via Hunter's, duct via ASHRAE friction chart + Huebscher.

Install (free tier, 30 seconds)

Have uv installed? Add this to your Claude Desktop config and restart:

{
  "mcpServers": {
    "hvac": {
      "command": "uvx",
      "args": [
        "--from",
        "git+https://github.com/NightFast-app/hvac-mcp",
        "hvac-mcp"
      ]
    }
  }
}

Installs direct from GitHub. PyPI release coming soon — once live, you'll only need "args": ["hvac-mcp"].

Full client configs (Claude Code, Claude app, ChatGPT, Cursor) in docs/CONNECTING.md.
